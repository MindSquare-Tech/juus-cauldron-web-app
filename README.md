# Juus Cauldron Website

## Getting Started

Follow these steps to set up and run the project locally:

### 1. Install the Project Dependencies
After cloning the repo into your local system, go to the project directory and install the dependencies
```bash
npm install
```

### 2. Start the Project
After all the dependencies are installed run the project using the following command
```bash
npm run dev
```
Your project will be running on the 5173 port
